# SpiderGUI

This is a simple modular graphical endpoint for the [Scrapy](https://scrapy.org/) python web-crawler library.
It allows to issue crawls with a basic list of urls and prefix-blacklist and supports several post-processing and analytical features, heavily based on [pandas](http://pandas.pydata.org/).
